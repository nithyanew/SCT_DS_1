# SCT_DS_1
## Data Visualization using Matplotlib
This project demonstrates basic data visualization using Python's Matplotlib library.
It includes:
Bar Chart – Gender Distribution
Histogram – Age Distribution
# Project Overview
This repository contains two simple visualization examples:
Gender Distribution (Bar Chart)
Displays the number of people categorized by gender.
Age Distribution (Histogram)
Shows how ages are distributed across different ranges using bins.
# Technologies Used
Python
Matplotlib
# Visualization 1: Gender Distribution (Bar Chart)
Description
This bar chart represents the count of people in three categories:
Male
Female
Other
Output
X-axis → Gender
Y-axis → Number of People
Clearly shows category comparison.
# Visualization 2: Age Distribution (Histogram)
This histogram shows the frequency distribution of ages grouped into 6 bins.
Output
X-axis → Age groups
Y-axis → Frequency
Displays how ages are spread across ranges.
